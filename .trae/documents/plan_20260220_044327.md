# 校园图书共享 - 功能添加方案

## 功能清单

1. **搜索功能** - 按书名/作者搜索图书
2. **个人中心** - 查看我的帖子、编辑个人信息
3. **帖子编辑/删除** - 修改和删除自己发布的帖子
4. **图片上传** - 支持上传图书封面图片

***

## 详细实现方案

### 一、搜索功能

#### 后端修改

**文件：** `server/controller/PostController.go`

* 修改 `GetPostList` 函数，支持关键词搜索参数

* 使用 `LIKE` 查询匹配书名和作者

#### 前端修改

**文件：** `android/app/src/main/java/com/example/campus_book_share/MainActivity.kt`

* 顶部添加搜索框（EditText）

* 添加搜索按钮或监听输入变化

* 调用 API 时传递 keyword 参数

***

### 二、个人中心

#### 后端新增/修改

1. **新建/修改：** `server/model/user.go` - 已有 User 模型，无需大改
2. **修改：** `server/controller/UserController.go`

   * 添加 `GetUserInfo` 接口 - 获取当前用户信息

   * 添加 `UpdateUserInfo` 接口 - 更新用户信息
3. **修改：** `server/controller/PostController.go`

   * 添加 `GetMyPosts` 接口 - 获取当前用户发布的帖子
4. **修改：** `server/routes/routes.go` - 添加新路由

#### 前端新增

1. **新建：** `android/app/src/main/java/com/example/campus_book_share/ProfileActivity.kt` - 个人中心页面
2. **新建：** `android/app/src/main/res/layout/activity_profile.xml` - 个人中心布局
3. **修改：** `android/app/src/main/AndroidManifest.xml` - 注册新 Activity
4. **修改：** `android/app/src/main/java/com/example/campus_book_share/network/ApiService.kt` - 添加新 API 接口
5. **修改：** `android/app/src/main/java/com/example/campus_book_share/MainActivity.kt` - 添加个人中心入口

***

### 三、帖子编辑/删除

#### 后端修改

1. **修改：** `server/controller/PostController.go`

   * 添加 `UpdatePost` 接口 - 更新帖子（需验证作者）

   * 添加 `DeletePost` 接口 - 删除帖子（需验证作者）
2. **修改：** `server/routes/routes.go` - 添加新路由

#### 前端新增/修改

1. **新建：** `android/app/src/main/java/com/example/campus_book_share/EditPostActivity.kt` - 编辑帖子页面
2. **新建：** `android/app/src/main/res/layout/activity_edit_post.xml` - 编辑帖子布局
3. **修改：** `android/app/src/main/java/com/example/campus_book_share/DetailActivity.kt` - 添加编辑/删除按钮（仅发布者可见）
4. **修改：** `android/app/src/main/java/com/example/campus_book_share/network/ApiService.kt` - 添加编辑/删除 API
5. **修改：** `android/app/src/main/AndroidManifest.xml` - 注册 EditPostActivity

***

### 四、图片上传功能

#### 后端新增/修改

1. **新建：** `server/controller/UploadController.go` - 图片上传控制器
2. **修改：** `server/main.go` - 配置静态文件服务
3. **修改：** `server/routes/routes.go` - 添加上传路由
4. **创建目录：** `server/uploads/` - 存放上传的图片

#### 前端修改

1. **修改：** `android/app/src/main/AndroidManifest.xml` - 添加存储权限
2. **修改：** `android/app/src/main/java/com/example/campus_book_share/PublishActivity.kt` - 添加图片选择和上传功能
3. **修改：** `android/app/src/main/java/com/example/campus_book_share/network/ApiService.kt` - 添加上传 API（Multipart）
4. **修改：** `android/app/src/main/java/com/example/campus_book_share/network/RetrofitClient.kt` - 配置 Multipart 支持
5. **修改：** `android/app/build.gradle.kts` - 添加图片选择库依赖（如 Glide）

***

## 实现顺序建议

1. 搜索功能（最简单，先试水）
2. 个人中心（核心功能）
3. 帖子编辑/删除（完善交互）
4. 图片上传（较复杂，最后做）
