# Campus Book Share - Recommendation Service

基于 TF-IDF + 余弦相似度 的相似图书推荐服务

## 环境要求

- Python 3.8+

## 安装依赖

```bash
cd python
pip install -r requirements.txt
```

## 运行测试

```bash
cd python
python test_recommendation.py
```

## 启动 API 服务

```bash
cd python
python app.py
```

服务运行在 http://localhost:5000
